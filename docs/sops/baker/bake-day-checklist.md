# Quick Reference: Bake Day Checklist

Use this on pickup or delivery day.

## Start Of Day

- [ ] Open Orders.
- [ ] Review paid orders for this week.
- [ ] Look for customer notes and dietary notes.
- [ ] Check for any order marked as an issue.

## Pickup Orders

- [ ] Open Pickup.
- [ ] Check the pickup overview.
- [ ] Review the pickup queue.
- [ ] Mark orders Ready when they are packed, if you use that step.
- [ ] Send pickup notes when orders are truly ready, if you use them.
- [ ] Mark Picked up only after the customer has the order.

## Delivery Orders

- [ ] Open Deliveries.
- [ ] Check delivery addresses and phone numbers.
- [ ] Fix missing details before leaving.
- [ ] Review the delivery plan.
- [ ] Send delivery notes when you are heading out, if you use them.
- [ ] Mark delivered only after drop-off.

## Customer Updates

- [ ] Keep messages short and clear.
- [ ] Check that you picked the right customer.
- [ ] Check the message log after sending.

## Watch For Issues

- [ ] Missing address.
- [ ] Missing phone or email.
- [ ] Customer note that changes packing.
- [ ] Item that cannot be fulfilled.
- [ ] Customer says they paid but you do not see the order.

## If Something Feels Wrong

Pause. Do not change paid orders casually. Ask Chandler for help if the answer is not clear.
