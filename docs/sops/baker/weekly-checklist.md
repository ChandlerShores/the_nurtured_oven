# Quick Reference: Weekly Checklist

Use this to move through a normal bakery week.

## Before Opening Orders

- [ ] Update this week's menu.
- [ ] Check item names, prices, descriptions, and photos.
- [ ] Choose what is live or featured.
- [ ] Mark sold-out items if needed.
- [ ] Check the public menu.
- [ ] Open ordering only when everything looks right.

## While Orders Are Open

- [ ] Check paid orders once or twice a day.
- [ ] Look for customer notes or dietary notes.
- [ ] Watch for items selling faster than expected.
- [ ] Close ordering if the week starts to feel too full.

## Before Ordering Closes

- [ ] Review paid orders.
- [ ] Check pickup and delivery counts.
- [ ] Check for missing delivery details.
- [ ] Close ordering if you need to stop taking more orders.

## Prep Day

- [ ] Review paid orders.
- [ ] Check item totals.
- [ ] Check pickup orders.
- [ ] Check delivery orders and addresses.
- [ ] Mark issue orders so they are easy to find.

## Bake And Fulfillment Day

- [ ] Use the pickup queue for pickup orders.
- [ ] Use the delivery page for delivery orders.
- [ ] Send customer updates if you use them.
- [ ] Mark orders picked up or delivered after they are complete.

## After Fulfillment

- [ ] Check that active pickup and delivery lists are clear.
- [ ] Review any issue orders.
- [ ] Note anything to fix before the next menu drop.

## If You Are Unsure

Close ordering first. Then check the public menu and ask Chandler for help.
