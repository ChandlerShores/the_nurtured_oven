# Training Plan: The Nurtured Oven Admin

Use this plan for Chandler and Kori to practice the weekly flow together.

## Session 1: Core Weekly Workflow

Time: 45 to 60 minutes

Goal: Kori can update the weekly menu and control ordering.

Practice:

- Open the admin area.
- Update one menu item.
- Choose live and featured settings.
- Save changes.
- Check the public menu.
- Open and close ordering.

Afterward, Kori should be able to:

- Update the weekly menu with confidence.
- Check what customers see.
- Close ordering if anything feels wrong.

## Session 2: Bake Day Workflow

Time: 45 to 60 minutes

Goal: Kori can review paid orders and work through pickup and delivery day.

Practice:

- Open Orders and review the order list.
- Check customer notes.
- Open Pickup and review the queue.
- Open Deliveries and review the delivery plan.
- Mark a practice order picked up or delivered only after it is complete.

Afterward, Kori should be able to:

- Find what needs to be baked and packed.
- Track pickup and delivery orders.
- Avoid changing paid orders casually.

## Session 3: Customer Updates And Extras

Time: 30 to 45 minutes

Goal: Kori can send simple customer updates and know what to ignore until needed.

Practice:

- Open Messages.
- Search for a customer order.
- Preview a short update.
- Check the message log.
- Review the emergency guide.

Afterward, Kori should be able to:

- Send clear updates when needed.
- Know when to ask for help.
- Start small and use only the areas she needs each week.
