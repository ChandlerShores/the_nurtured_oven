# Project

Minimal [Next.js](https://nextjs.org/) 14 (App Router) + TypeScript + Tailwind CSS scaffold.

## Commands

```bash
pnpm install
pnpm dev
pnpm build
pnpm lint
```

## Structure

- `app/` — routes and layouts (`layout.tsx`, `page.tsx`, `globals.css`)
- `public/` — static assets (add files as needed)

Path alias `@/*` is configured in `tsconfig.json` for future `components/`, `lib/`, etc.
